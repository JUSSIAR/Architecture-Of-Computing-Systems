# HW1

> ДЗ1 по предмету АВС

### Setup
1. > Для сборки исполняемого файла необходимо выполнить:

```
mkdir build
cd build
cmake ..
make
```

2. > Для запуска необходимо выполнить (из build):

```
./hw1 {input_file} {output_file}
```

3. > При необходимости генерации больших тестов выполнить:

```
cd tests
make
```
